# Display-odd-numbers-1-100-
Display Odd Numbers (1-100)

public class OddNumbers {
    public static void main(String[] args) {
        System.out.println("Odd numbers between 1 and 100:");
        for (int i = 1; i <= 100; i += 2) {
            System.out.print(i + " ");
        }
    }
}

Output

Odd numbers between 1 and 100:  
1 3 5 7 ... 99
